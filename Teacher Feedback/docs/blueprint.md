# **App Name**: EduAI Insights

## Core Features:

- Response Analysis: Analyzes student responses and engagement levels to identify effective teaching strategies.
- Engagement Tips: Provides AI-driven personalized tips to enhance classroom engagement. The AI tool decides if specific engagement techniques will be recommended based on response analysis.
- Metric Dashboard: Displays key performance indicators (KPIs) for teachers: Participation rate, Understanding metrics and quantity/sentiment analysis of feedback.
- Summary Message: Presents an AI-generated summary statement that highlights impactful teacher actions.

## Style Guidelines:

- Primary color: Deep violet (#7950f2) to reflect intellect, knowledge, and modern education.
- Background color: Light violet (#f0ebfa), subtly desaturated for a comfortable viewing experience.
- Accent color: Soft pink (#f250ae) for highlighting important insights and calls to action, creating contrast without harshness.
- Font: 'PT Sans' (sans-serif), for both headlines and body text, to provide a modern but welcoming and easily readable look.
- Use simple, clean icons to represent data metrics and engagement suggestions.
- Emphasize clarity, prioritizing essential information with a clean, spacious layout.
